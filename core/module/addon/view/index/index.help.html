EXPORTER IMPORTER

Exporter produit une archive au nom du module contenant les pages concernées ainsi que les données et ressources utilisées par le module dans ces pages.

Vous pouvez vous en servir comme d'une sauvegarde partielle ou pour transférer les pages et les données du module vers un autre site.

Une fois le module installé l'import permet de restaurer les pages et les données sauvegardées. Vous avez au préalable transféré le fichier zip d'un export sur votre serveur par 'Gérer les fichiers'. Si une page de même nom existe sur votre site vous serez invité à modifier son nom.
